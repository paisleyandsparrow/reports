# Creator Coders Social Content Schedule (Designer Version)

## Purpose
This is the single approval-ready content calendar for social, optimized for still imagery.
Each row includes:
- Working title
- Subject/focus
- Content direction (what to say)
- Exact designer deliverables

## Scope Update (Approved)
- YouTube removed from this plan.
- Creative direction is still-first.
- Optional reels are allowed, but built from designed still assets (no custom video production required).

## Brand Guardrails
- Brand line: Software for serious creators.
- Core CTA: Start 7 days free (no card required).
- Destination URL: https://creatorcoders.com
- Product URL mention in copy: https://portal.creatorcoders.com/login
- Voice: direct, practical, anti-fluff, creator-to-creator.

## Still-First Creative System
- Primary outputs: carousels, single graphics, thread covers, story cards.
- Reels (optional): slideshow-style from stills only, light zoom/pan, text overlays.
- Color mood: warm neutral base, espresso text, pink accent.
- Visual motifs: dashboard snippets, creator workflow, before/after comparison, simple ROI numbers.
- Typography: elegant serif headline + clean sans body (match website aesthetic).

## Master Asset Specs (Designer)
- Instagram carousel: 1080x1350
- Instagram story: 1080x1920
- Instagram feed static: 1080x1350
- X graphic/thread cover: 1600x900
- Safe text margins: keep text 8 to 10 percent inside all edges
- Export: PNG for static, JPG acceptable for photo-heavy slides
